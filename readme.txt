Show textbox, where do you want.

== Description ==

This a textbox as like wordpress text widget. But this widget can hideable or showable where do you want.
This mean this widget have settings as like show home, show page etc. This widgets have support shortcodes.

= For Turkish (Türkçe İçin) =
http://halil.me/hal-html-widget.html

== Installation ==

1. Extract to arhive.
2. Upload `hal_html_widget.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Now find `Hal Html Widget` on `Theme -> Widgets` page. Drop down this and write text or html codes and select show where do you want.
